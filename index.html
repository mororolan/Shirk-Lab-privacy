📋 Introduction

Thank you for using the "Fish Slacking Quantifier" Chrome extension. We take your privacy very seriously. This privacy policy explains how we handle your information.

📊 Data Collection

What data do we collect?

None. This extension does not collect, transmit, or store any user data to external servers.

Locally Stored Data

This extension only stores the following information in your local browser:

Data Type Purpose Storage Location
💰 Monthly Salary Calculate daily/hourly wage Local Browser
⏰ Work Hours Calculate work progress Local Browser
🐟 Slacking Records Track slacking time Local Browser
📋 Overtime Records Track overtime hours Local Browser
🔔 Wooden Fish Merit Record tap counts Local Browser
This data:
  • Is stored only on your device
  • Is never sent to any server
  • Is not shared with any third parties
  • Can be deleted by uninstalling the extension

🌐 Data Transmission

This extension:

  • Does not make any network requests
  • Does not connect to any external servers
  • Does not use any analytics or tracking services
  • Does not load any remote code

🔐 Permissions

This extension only requests the storage permission to:

  • Save your settings in the local browser
  • Save your records in the local browser

We do not request any other permissions, including:

  • Access to your browsing history
  • Access to webpage content
  • Access to your personal information

🔗 Third-Party Services

This extension does not use any third-party services, including:

  • Analytics services (e.g., Google Analytics)
  • Advertising services
  • Social media integrations
  • Cloud storage services

🛡️ Data Security

Since all data is stored in your local browser, data security depends on your device's security. We recommend:

  • Keep your browser and OS updated
  • Use strong passwords to protect your device
  • Avoid storing sensitive information on public devices

👶 Children's Privacy

This extension is not intended for children under 13 and does not knowingly collect any information from children.

If we make any changes to this privacy policy, we will post the updated policy on this page and update the "Last Updated" date.
